BlitzTips — Real-Time Trafficking Tip Platform

A secure, real-time investigation platform built at John Jay College to collect, triage, and visualize human-trafficking tips. The system pairs a .NET Core/API backend with a React-style admin UI to deliver live maps, investigator dashboards, and exportable intelligence feeds — purpose-built for handling sensitive PII and operational data.

Integrates Google Maps API for geospatial insight, SignalR for push updates, and Hangfire for background jobs at scale. Security is enforced end-to-end with OAuth2 (Azure AD), TLS in transit, field-level encryption at rest, and role-based access controls, backed by comprehensive audit trails and anomaly-detection routines. Modular Razor/Bootstrap views enable fast filtering, drill-downs, and data exports across demographics, vehicles, and observations — empowering stakeholders with actionable, trend-level insight.
